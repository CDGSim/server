        Introduction

        Le système de logs numériques permet l'édition et la consultation des logs de simulations à CDG de façon dématérialisée.

        Ce système a été dévelopé dans l'optique de réduire l'utilisation de papier et d'encre d'impression, et de faciliter la création et la mise à jour des logs de simulations.

        Il s'articule autour de deux composants : un éditeur de logs et un server délivrant un client web.

        Serveur

        Le serveur permet la consultation des logs depuis un navigateur web. Il héberge les fichiers de logs et de simulation, analyse leur contenu et les présente sous forme de pages HTML.

        Éditeur de logs

        Les fichiers de logs sont des fichiers texte, formattés au format JSON, avec une extension .simlog.

        Ces fichiers peuvent ainsi être édités avec n'importe quel éditeur de texte. Ceci permet d'apporter des modifications en masse via une commande rechercher/remplacer globale, par exemple lorsque les simulations sont mises à jour pour modifier un type d'appareil qui n'est plus utilisé, ou supprimer une compagnie aérienne qui n'existe plus.

        Ces fichiers peuvent également être édités via des applications dédiées :

        • une application web
        • une application macOS, comportant quelques fonctionnalités avancées
